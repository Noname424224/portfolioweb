(03) обо мне

Что я
пишу.

Работаю на стыке веба, Python и систем — и довожу проекты до рабочего продукта в проде, а не до красивого макета.

В вебе собираю интерфейсы на React и чистом JS, верстаю на Tailwind, подключаю Firebase Firestore для реал-тайм-данных, авторизации и админ-панелей. Спокойно довожу проект до деплоя на Vercel.

В Python пишу телеграм-ботов, автоматизацию и голосовые пайплайны: распознавание речи и синтез, настольные приложения с графическим интерфейсом. Отдельный интерес — локальные LLM: разворачиваю модели через Ollama и встраиваю их в приложения так, чтобы всё работало офлайн на своём железе.

Не боюсь незнакомых стеков: писал мод на Java под Forge с собственной AI-сущностью, собирал кроссплатформенный VPN-клиент на Flutter поверх Xray-core, настраивал Linux с тайловым оконным менеджером. Чем техничнее и нетривиальнее задача, тем интереснее.

/ веб

React, чистый JavaScript, Tailwind, адаптивная вёрстка mobile-first, деплой на Vercel.

/ данные

Firebase Firestore — реал-тайм-синхронизация, авторизация, мультиаккаунтные админки.

/ python

Телеграм-боты, автоматизация, голосовые пайплайны (Whisper, Edge TTS), десктоп на CustomTkinter.

/ llm

Локальный инференс через Ollama, офлайн-модели на своём GPU, интеграция в приложения.

/ кросс-платформа

Flutter под Windows и Android, прототипирование интерфейсов, сборка под несколько целей.

/ системы

Java / Forge-моддинг, Linux и тайловые WM, сетевые протоколы, Xray-core (VLESS / Reality).
